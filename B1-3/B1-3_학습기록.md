# Make와 Zapier를 사용한 자동화

## 전체 흐름
- 구글 설문 -> 취미에 사냥이 들어있는지 여부에 따라 구글 시트 작성

## Make

- 자동화 실행 주기가 15분인 것으로 확인
- 자동화 블럭의 위치를 자유롭게 이동할 수 있음
- UI가 예뻐서 그런지 사용하기는 Make가 편한듯 보임
- 분기 처리를 If-else나 Router로 함
- 현재 시간이 미국 시간을 기준으로 작성되는 이슈가 있음 \<- 계정 설정의 Timezone 설정해도 안됨, {{addHours(var.scenario.executionStartedAt; 9)}} 형태의 명령어를 사용하면 됨

## Zapier

- 자동화 실행 주기가 거의 실시간으로 확인
- 자동화 블럭의 위치를 이동할 수 없음
- 분기 처리를 Path로 함
- 구글 시트에 헤더가 없는 경우 해당 칸을 감지하고 특정값을 입력할 수 없는 것으로 보임  
- 구글 시트 수정 후 테스트 실행을 해야 구글 시트의 변경이 감지되는 것으로 확인됨   
- 현재 시간이 미국 시간을 기준으로 작성되는 이슈가 있음 \<- 계정 설정의 Timezone 설정하면 됨
