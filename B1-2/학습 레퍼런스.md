## 1. 미션 소개
- `이번 미션의 목표는 기획 의도가 프롬프트를 거쳐 최종 영상으로 구현되기까지의 전체 제작 파이프라인을 스스로 설계하는 데 있습니다.` -> 만약 B1-1을 학습하지 않고 해당 과제부터 학습하는 경우 B1-1이 프롬프트 관련 과제이므로 먼저 학습하도록 안내
- 실제 브랜드를 사용하여 과제 진행시 `6. 제약 사항`에 안내된 부분처럼 저작권 이슈를 고려하도록 안내 

## 2. 최종 결과물
- 광고 영상 파일 길이가 10초 이내로 나오는데 `4. 기능 요구 사항`에서는 `최종 영상은 초 이상 60초 이내로 제작한다.`라고 나오는 부분 해결 필요

## 4. 기능 요구 사항
- 네이토 영상 생성의 경우 사운드가 포함되어 출력되긴 하나 오디오 생성 AI를 사용해보는 것이 과제 목표이므로 오디오 생성 AI를 사용해보도록 유도하는 것이 좋을 것 같음

## 6. 제약 사항
- `Premiere Pro`의 경우 무료로 사용 불가하고 별도로 설치해서 사용해야 하여 사용 불가함
- [ControlNet](https://github.com/lllyasviel/controlnet)이나 [Ip-Adapter](https://github.com/tencent-ailab/IP-Adapter)에 대한 개념을 학습하도록 유도하면 좋을 것 같음
- [Stable Diffusion](https://stability.ai/stable-image)에 대해 학습해보도록 안내해도 좋을 것 같음. [참고자료2](https://wikidocs.net/234116) 
- [WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)나 [CompyUI](https://comfy.org/), [Diffusers](https://huggingface.co/docs/diffusers/ko/index) 등 이미지 생성 AI 소프트웨어/라이브러리를 소개하는 것도 좋을 것 같음
